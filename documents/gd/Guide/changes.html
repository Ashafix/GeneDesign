  • Version History
Changes from 2.0 to 3.0
  • Changed vocabulary - oligo design is now building block creation; silent insertion / silent removal is now addition / subtraction
  • All code more efficient. Suffix tree implementation for restriction enzyme addition searching
  • Building Block Design
    • Three modes for design now - restriction enzyme site overlap, uracil excision, and constant length sequence overlap
    • output as FASTA or tab-delimited files
  • Revamped Enzyme List and Enzyme Selection Criteria
    • Updated enzyme list to reflect commercial availability and 2009 pricing
    • Added overhang palindromy, buffer, heat inactivation temperature, incubation temperature, and star activity as criteria.
    • Added isoschizomers with different incubation temperatures.
  • Added B. subtilis and D. melanogaster to list of known organisms
  • Codon Juggling
    • Added "Least Different RSCU" algorithm
    • Added RSCU graph to output
  • Restriction Enzyme Subtraction now uses "Least Different RSCU" algorithm
  • Reverse Translation now reads FASTA input
  • Codon Bias Graphing
  • RSCU Value Generation
  • Ambiguous Nucleotide Translation
Changes from 1.9 to 2.0
  • Revamped Enzyme List and Enzyme Selection Criteria
    • New list composed of prototypes (and their neoschizomers) that are commercially available, single cutting, Type IIs with known cleavage sites.
    • No longer use buffer, vendor, and NEB availability as criteria.
    • Added incubation temperature, methylation sensitivity, star activity and exhaustive vendor list.
    • Now able to exclude enzymes with cleavage external to recognition site.
  • Added C. elegans to list of known organisms
  • Building Block Design now automatically excludes blunts, 1bp overhangs, and enzymes that cut outside of recognition site.
  • Codon Juggling detailed
    • Now warns if you are not using a simple coding sequence
    • Will allow you to not define an organism; hides optimization algorithms and randomizes most different
    • Define next most optimal and random as highly experimental algorithms
  • Restriction Site Addition detailed
    • Allow reconsideration of automatic site selection
    • Fixed error messages about simple coding sequences
  • Miscellaneous bug fixes that you probably never noticed
  • Manual updated accordingly and PDF version added.
  • Now with version history!
